
# Learning Streak Tracker

A small web app I built while learning to code.

I'm a CSE student just getting started, and I noticed that the hardest part
of learning isn't the hard topics, it's showing up every single day without
skipping. So I made this little tracker to keep myself honest.

You write down what you want to learn today (like "watch 1 Python video" or
"build a Flutter button"), tick things off as you finish them, and the app
keeps a streak of how many days in a row you actually showed up.

## What it does

- Add things you want to learn today
- Tick them off when done (or untick if you made a mistake)
- Delete anything you don't need
- Keeps a daily streak counter
- Remembers everything even after you close the tab

## Built with

- HTML
- CSS
- JavaScript

No frameworks yet. I wanted to understand every line myself before reaching
for bigger tools.

## How to run it

1. Download or clone this repository.
2. Open `index.html` in any web browser.

That's it, no installation needed.

## What I learned building this

- How to save data in the browser with `localStorage`
- How to add and remove items from a list with JavaScript
- How to keep my colors and styles organized so they're easy to change
